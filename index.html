About Me

Olá, sou Felipe, um Analista de T.I. Sênior com experiência em tecnologia e um forte foco em nuvem. Atuo na interseção entre nuvem e desenvolvimento, com certificações em Azure 900, IA 900 e AWS Practitioner. Minha experiência com a AWS inclui gerenciamento de serviços como Amazon S3, EFS, EBS e Amazon VPC, aplicando boas práticas de segurança e eficiência. Além disso, possuo conhecimento em Azure AD, Azure Blob Storage e Azure File Storage, ampliando minha experiência em nuvem. Sou proficiente no uso do Identity and Access Management (IAM) e Amazon CloudWatch para monitorar e otimizar as operações na nuvem. Tenho habilidades sólidas em Javascript, HTML e CSS, que são fundamentais para o desenvolvimento web e para a criação de interfaces de usuário interativas e responsivas. Além disso, sou competente em Figma, uma ferramenta essencial para o design de interfaces digitais. Meu objetivo é migrar estrategicamente para o desenvolvimento de software, combinando minha experiência em T.I. com a paixão pela nuvem para criar soluções inovadoras. Habilidades: AWS, Azure, Microsoft 365, Active Directory, ServiceNow, Cyber Security, Redes, Javascript, Figma, HTML, CSS.
